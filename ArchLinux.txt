Window Manager

  grub-customizer
  Install Light DM
    Enable lightdm.service
    /etc/lightdm/lightdm.conf should have greeter-session=lightdm-gtk-greeter
    /etc/lightdm/lightdm-gtk-greeter.conf to change background image
  GTK Theme: Adwaita
  Icon Theme: Gnome-Brave
  dm-tool switch-to-greeter can be used to lock
  Disable netctl and Enable NetworkManager
  Enable Gnome Services during startup
